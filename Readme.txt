# 🌦️ Weather Forecast WINDY

This is a simple, responsive Weather Forecast Web Application built using **HTML**, **CSS**, and **JavaScript**. The app utilizes the **Fetch API** to interact with the **OpenWeatherMap API** and provides real-time weather updates, 3-day forecasts, and common weather data for major cities.

---

## 🔧 Features

- 🌍 **Search by CitY Or Country Name**  
  Users can enter any city Or Country name to get current weather data and a 3-day forecast.

- 🧠 **JavaScript Concepts Covered**
  - DOM Manipulation
  - Event Handling
  - Functions and Objects
  - API Integration using `fetch()`
  - Conditional Rendering
  - Template Literals
  - Async/Await and Promises

- ☁️ **Live Weather Data**  
  Fetches and displays:
  - Temperature
  - Wind Speed
  - Humidity
  - Weather Condition (Icon + Description)
  - Time and Date (converted from UNIX)

- 📅 **3-Day Forecast**
  Shows upcoming weather trends for the next three days from the searched location.

- 🏙️ **Fixed City Display**
  Displays current weather data for:
  - New Delhi
  - Kolkata
  - Mumbai

- 🎨 **Responsive UI**
  Designed with modern CSS for a clean, minimal and mobile-friendly layout.

---

## 📦 Technologies Used

- **HTML5** – for structuring the layout.
- **CSS3** – for styling, layout and responsive design.
- **JavaScript (Vanilla)** – for all interactivity and logic.
- **OpenWeatherMap API** – to fetch weather data.
